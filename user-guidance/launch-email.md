## What The Tool Does

The tool builds malaria budgets from SNT-style intervention scenarios and unit cost assumptions. It supports visualisations, budget comparisons, and Excel exports for the currently included intervention areas:

- Mass ITN campaign
- Routine / continuous ITN
- Indoor residual spraying
- Seasonal malaria chemoprevention
- IPT for school-age children
- Malaria vaccine
- IPTp in pregnancy

The current version does not support case management, other interventions, or programme/activity cost categories that are not represented in the uploaded/default unit cost file.

## First Things To Try

1. Sign in and reset your password if prompted.
2. Open the Budget generation tab.
3. Review the existing scenario and cost set.
4. Generate a budget or open an existing budget from the library.
5. Open Budget visualisation to review charts and tables.
6. Export Excel if you need a line-by-line copy of the budget.

## Saving And Shared Use

The app shares saved scenarios, cost sets, and generated budgets across authorised users.

Before closing your browser, please wait until the top-right status says `Shared data saved`.

If you open the app in another browser, refresh the page and wait for `Shared data loaded`.

Please only use `Sync now` if asked during troubleshooting, or if you are sure that the browser you are using contains the budget library that should be preserved.

The app is shared, but it is not real-time co-editing like Google Docs. Please avoid two people editing the same scenario or cost set at the same time.
